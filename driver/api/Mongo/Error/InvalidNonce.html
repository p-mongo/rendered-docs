Overview

This exception is raised when the server nonce returned does not start with the client nonce sent to it.

def initialize(nonce, rnonce)
  @nonce = nonce
  @rnonce = rnonce
  super("Expected server rnonce '#{rnonce}' to start with client nonce '#{nonce}'.")
end
